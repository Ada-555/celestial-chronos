# Celestial Chronos ⌚

**🔗 Live:** https://ada-555.github.io/celestial-chronos/

---

A high-fidelity 3D web experience presenting a forensic-quality deconstruction of a luxury mechanical watch. Built for the discerning collector who demands both precision engineering and visual spectacle.

## Features

- ⚙️ **3D Watch Deconstruction** — Scroll to explode/assemble 200+ individual components
- 🎨 **Cyberpunk Gold + Neon Cyan** — Rose gold (0xB08D57) with emissive accents
- 💎 **Realistic Materials** — Ruby jewels, sapphire crystal, gold plating
- ⏱️ **Live Clock Hands** — Real-time hours, minutes, seconds
- ⚡ **Balance Wheel Oscillation** — Noise-driven animation
- 🌟 **Selective Bloom** — Only neon elements glow
- 🎞️ **Film Grain** — Subtle analog texture
- 🔥 **Auto-Rotating Camera** — Hands-free viewing

## Tech Stack

- Three.js (r160)
- Custom transmission materials
- Post-processing (UnrealBloomPass + film grain)
- GitHub Pages
